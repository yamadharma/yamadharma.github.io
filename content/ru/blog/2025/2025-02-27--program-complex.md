Программный комплекс.

<!--more-->

{{< toc >}}


## <span class="section-num">1</span> Общая информация {#общая-информация}

-   Программный комплекс есть набор взаимосвязанных программ, которые работают вместе для решения определённых задач.
-   Программный комплекс отличается от простого набора программ своей целостностью и согласованностью работы всех компонентов.
-   Каждая программа в комплексе специально разработана или адаптирована для взаимодействия с другими программами этого комплекса.


## <span class="section-num">2</span> Основные характеристики {#основные-характеристики}


### <span class="section-num">2.1</span> Согласованность по функциям {#согласованность-по-функциям}

-   Все программы в комплексе дополняют друг друга
-   Нет функциональных пересечений
-   Каждая программа выполняет свою конкретную задачу


### <span class="section-num">2.2</span> Единообразие форматов {#единообразие-форматов}

-   Единый формат данных
-   Согласованный формат ввода/вывода
-   Стандартизированные протоколы обмена информацией


### <span class="section-num">2.3</span> Четкие интерфейсы {#четкие-интерфейсы}

-   Точно определенные точки взаимодействия
-   Стандартизированные способы передачи данных
-   Продуманная система взаимодействия компонентов


### <span class="section-num">2.4</span> Целостность решения {#целостность-решения}

-   Комплекс решает конкретную большую задачу
-   Все компоненты работают как единое целое
-   Результат работы одного компонента может использоваться другим
