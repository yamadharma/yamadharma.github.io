## <span class="section-num">1</span> Общая информация {#общая-информация}

-   Раскладка Miryoku  --- эргономичная и минималистичная клавиатурная раскладка, разработанная для компактных клавиатур (обычно с 36–42 клавишами).
-   Поддерживает различные прошивки (QMK, ZMK, KMK).


### <span class="section-num">1.1</span> Основные особенности {#основные-особенности}

-   Эргономика: Клавиши расположены так, чтобы минимизировать движение пальцев, снижая нагрузку на руки.
-   Универсальность: Совместима с разными типами клавиатур и прошивками.
-   Многослойность: Использует слои для доступа к дополнительным функциям (цифры, символы, навигация) без увеличения количества клавиш.
-   Документация: Подробные руководства и примеры конфигурации.


### <span class="section-num">1.2</span> Плюсы {#плюсы}

-   Подходит для долгой работы за клавиатурой.
-   Экономия места за счёт компактного дизайна.
-   Активное сообщество и регулярные обновления.


### <span class="section-num">1.3</span> Минусы {#минусы}

-   Требует времени для адаптации.
-   Ограниченная совместимость с некоторыми играми или специализированным ПО.


## <span class="section-num">2</span> Слои в Miryoku {#слои-в-miryoku}

-   В Miryoku  используется система слоёв, которые позволяют компактно разместить все необходимые функции на небольшой клавиатуре.


### <span class="section-num">2.1</span> **Base (Базовый)** {#base--базовый}

-   Назначение: Основной слой для ввода букв (по умолчанию — Colemak-DH, но поддерживает QWERTY, Dvorak и другие).
-   Особенности: Включает пробел, Backspace и клавиши-модификаторы (Shift, Ctrl и т.д.) на домашнем ряду.


### <span class="section-num">2.2</span> **Nav (Навигация)** {#nav--навигация}

-   Активация: Удержание клавиши левым большым пальцем.
-   Содержит: Клавиши для перемещения (←↑↓→), Page Up/Down, Home/End, Tab.


### <span class="section-num">2.3</span> Mouse (Мышь) {#mouse--мышь}

-   Активация: Удержание клавиши правым большим пальцем.
-   Содержит: Управление курсором, клики (левый/правый), скроллинг.


### <span class="section-num">2.4</span> Media (Медиа) {#media--медиа}

-   Активация: Комбинация клавиш большого пальца.
-   Содержит: Регулировка громкости, пауза/воспроизведение, управление яркостью экрана.


### <span class="section-num">2.5</span> Num (Цифры) {#num--цифры}

-   Назначение: Ввод чисел и математических операций (+, -, \\\*, /).
-   Особенности: Расположение цифр оптимизировано под частоту использования (например, 1–5 на левой руке, 6–0 на правой).


### <span class="section-num">2.6</span> Sym (Символы) {#sym--символы}

-   Содержит: Скобки, кавычки, знаки препинания (например, `!`, `@`, `#`), а также символы для программирования (например, `{ }`, `[ ]` ).


### <span class="section-num">2.7</span> Fun (Функциональные клавиши) {#fun--функциональные-клавиши}

-   Назначение: Доступ к F1-–F12, Print Screen, Pause/Break.


### <span class="section-num">2.8</span> Adjust (Настройки) {#adjust--настройки}

-   Активация: Комбинация клавиш обоими большими пальцами.
-   Содержит: Переключение между раскладками (Colemak/QWERTY), сброс клавиатуры, активацию режима DFU для прошивки.
