abstract: Программа применяется для обеспечения учебного процесса по курсу «Параллельное
  программирования» для 4 курса бакалавриата. Программа включает модули с интерфейсами
  функций и подпрограмм, которые должны быть дополнены студентами в соответствии с
  заданием лабораторных работ. Программа содержит make-файлы для сборки всех компонентов
  и python-скрипты для визуализации замеров времени, ускорения и эффективности параллелизации.
  Программа предусматривает возможность проверить правильность реализаций функций
  учащимися с помощью набора тестов.
