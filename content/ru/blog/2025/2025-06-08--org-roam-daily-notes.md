## <span class="section-num">1</span> Общая информация {#общая-информация}

-   Org-roam Daily Notes --- функция для создания ежедневных заметок в Emacs с использованием Org-roam.


## <span class="section-num">2</span> Основные возможности {#основные-возможности}

-   Автоматическое создание ежедневных заметок
-   Интеграция с системой ссылок Org-roam
-   Возможность настройки расположения файлов


## <span class="section-num">3</span> Преимущества использования {#преимущества-использования}

-   Централизованное хранение ежедневных заметок
-   Автоматическая генерация структуры файлов
-   Простая навигация по записям
-   Интеграция с системой заметок Org-roam


## <span class="section-num">4</span> Настройка {#настройка}

-   По умолчанию файлы ежедневных заметок создаются в корневой директории org-roam.
-   Их можно переместить в отдельную папку:
    ```elisp
    (setq org-roam-dailies-capture-templates
          '(("d" "daily" plain
             (function org-roam-capture--get-point)
             ""
             :immediate-finish t
             :file-name "dailies/%<%Y-%m-%d>"
             :head "#+TITLE: %<%Y-%m-%d>")))
    ```


## <span class="section-num">5</span> Сочетания клавиш {#сочетания-клавиш}

-   Навигация и создание
    -   `C-c n d d` - открыть или создать сегодняшнюю ежедневную заметку
    -   `C-c n d n` - создать новую ежедневную заметку
    -   `C-c n d p` - открыть предыдущую ежедневную заметку
    -   `C-c n d n` - открыть следующую ежедневную заметку

-   Навигация по ссылкам
    -   `C-c C-o` - открыть ссылку под курсором
    -   `C-b` - переместиться на один символ назад (если ссылка не открывается)
    -   `RET` - если установлен параметр `org-return-follows-link`, можно переходить по ссылкам нажатием `Enter`

-   Дополнительные команды
    -   `SPC Enter` - альтернативное сочетание для перехода по ссылкам
    -   `M-x org-roam-dailies-find` - поиск и открытие любой ежедневной заметки
    -   `M-x org-roam-dailies-calendar` - открыть календарь с ежедневными заметками

-   В таблицах работает только стандартное сочетание `C-c C-o`
-   При создании новых ссылок может потребоваться перемещение курсора на один символ назад для корректного перехода
