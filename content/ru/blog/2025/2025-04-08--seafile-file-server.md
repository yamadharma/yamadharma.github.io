## <span class="section-num">1</span> Общая информация {#общая-информация}

-   Seafile --- это децентрализованное файловое хранилище с открытым исходным кодом.
-   Сайт: <https://www.seafile.com/>
-   Репозиторий: <https://github.com/haiwen/seafile>
-   Документация: <https://www.seafile.com/en/support/>
-   Клиенты: <https://www.seafile.com/en/download/>


### <span class="section-num">1.1</span> Основные возможности {#основные-возможности}

-   Хранение файлов любого типа и размера
-   Синхронизация между устройствами
-   История изменений файлов
-   Восстановление удаленных файлов
-   Совместный доступ к файлам
-   Интеграция с Git-репозиториями
-   Клиент для Windows, macOS и Linux
-   Мобильное приложение для iOS и Android


### <span class="section-num">1.2</span> Особенности работы {#особенности-работы}

-   Децентрализованная архитектура
-   Шифрование данных на стороне клиента
-   Дифференциальное резервное копирование
-   Групповая работа над проектами
