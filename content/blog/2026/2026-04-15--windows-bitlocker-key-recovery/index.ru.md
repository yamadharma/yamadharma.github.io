Windows. Ключ восстановления BitLocker.

<!--more-->

{{< toc >}}


## <span class="section-num">1</span> Общая информация {#общая-информация}

-   После сбоя компьютера Windows может запросить ключ восстановления BitLocker.
-   Снять защиту BitLocker и сохранить данные без ключа восстановления или пароля невозможно.


## <span class="section-num">2</span> Поиск ключа восстановления {#поиск-ключа-восстановления}

-   Обычно ключ хранится в учетной записи Microsoft.
-   Для идентификации запишите первые 8 цифр идентификатора ключа, которые видны на синем экране.
-   Перейдите на страницу ключей восстановления: <https://aka.ms/myrecoverykey>.
-   Войдите в свою учетную запись Microsoft (ту, что вы использовали на заблокированном компьютере).
-   В списке устройств выберите то, которое заблокировано.
-   Сопоставьте ключ по ID.
    -   Откройте раздел «Управление ключами восстановления».
    -   Сверьте первые 8 цифр идентификатора ключа восстановления.
    -   Ключ --- это 48-значный номер рядом с этим идентификатором.


## <span class="section-num">3</span> Отключение Bitlocker {#отключение-bitlocker}


### <span class="section-num">3.1</span> Через Панель управления {#через-панель-управления}

-   Подходит для всех версий Windows (10, 11, Pro, Enterprise).
-   Понадобятся права администратора.

-   Откройте _Панель управления_, перейдите в _Система и безопасность_ → _Шифрование диска BitLocker_.
-   Найдите нужный диск и нажмите _Отключить BitLocker_.
-   Подтвердите действие и дождитесь завершения расшифровки.


### <span class="section-num">3.2</span> Через «Параметры» {#через-параметры}

-   Для Windows 11 и Windows 10.
-   Если в _Панели управления_ вы видите пункт _Шифрование устройства_ вместо _Управление BitLocker_.
-   Откройте _Параметры_ (Win + I).
-   Windows 11. Перейдите в _Конфиденциальность и защита_ → _Шифрование устройства_.
-   Windows 10. Перейдите в _Обновление и безопасность_ → _Шифрование устройства_.
-   Установите переключатель в положение _Выключено_ и подтвердите действие.


### <span class="section-num">3.3</span> С помощью команд {#с-помощью-команд}

-   Командная строка:
    ```shell
    manage-bde -off C:
    ```
-   PowerShell:
    ```shell
    Disable-BitLocker -MountPoint "C:"
    ```


### <span class="section-num">3.4</span> Временно приостановить защиту {#временно-приостановить-защиту}

-   Можно временно приостановить защиту, а не отключать её полностью.
-   Это действие отключает проверку целостности системы до следующей перезагрузки.
-   Кнопка _Приостановить защиту_ находится рядом с кнопкой _Отключить BitLocker_ в _Панели управления_.
