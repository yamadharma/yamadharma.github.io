Linux. Дистрибутив Turnkey.

<!--more-->

{{< toc >}}


## <span class="section-num">1</span> Общая информация {#общая-информация}

-   Сайт: <https://www.turnkeylinux.org/>
-   Репозиторий: <https://github.com/turnkeylinux-apps/>
-   Набор подготовленных образов Linux с конкретными приложениями.
-   Сделан на основе Debian 12.
-   Поддерживается для установки на Proxmox (см. [Linux. Дистрибутив Proxmox]({{< relref "2024-06-04-linux-proxmox-distribution" >}})).


## <span class="section-num">2</span> Общая структура дистрибутивов {#общая-структура-дистрибутивов}

-   Предлагатся для скачивания iso-образы.
-   Образы включают в себя:
    -   web-панель Webmin для администрирования системы;
    -   подключенный SSH и AJAX web shell.


## <span class="section-num">3</span> Детали использования и вход в систему для администрирования {#детали-использования-и-вход-в-систему-для-администрирования}

-   Замените `12.34.56.789` действительным IP-адресом или именем хоста.


### <span class="section-num">3.1</span> Нет паролей по умолчанию {#нет-паролей-по-умолчанию}

-   По соображениям безопасности нет паролей по умолчанию.
-   Все пароли устанавливаются во время инициализации системы.


### <span class="section-num">3.2</span> SSL-сертификаты {#ssl-сертификаты}

-   Автоматически создаётся самоподписанный сертификат.
-   Через меню _Confconsole Advanced menu_ можно создать сертификат Let's Encypt SSL (см. [Провайдеры ACME]({{< relref "2022-05-02-acme-providers" >}})).
-   Соединения:
    -   `http://12.34.56.789/` : не зашифрованное соединение;
    -   `https://12.34.56.789/` — соединение зашифровано с помощью самоподписанного SSL-сертификата.


### <span class="section-num">3.3</span> Администрирование {#администрирование}

-   Администратор системы : `root`.
-   Web-подключение:
    -   `https://12.34.56.789:12321/` : панель управления системой;
    -   `https://12.34.56.789:12320/` : web-терминал командной строки.
-   ssh-клиент:
    ```shell
    ssh root@12.34.56.789
    ```
-   TUI конфигуратор:
    ```shell
    confconsole
    ```

    -   <https://www.turnkeylinux.org/docs/confconsole>
