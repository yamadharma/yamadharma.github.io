## <span class="section-num">1</span> Международный стандарт {#международный-стандарт}

-   Стандарт ISO 269 определяет несколько стандартных конвертов, предназначенных для вкладывания в них бумаги форматов, определённых в ISO 216.

| Формат | Размеры, мм | Подходит для формата бумаги     |
|--------|-------------|---------------------------------|
| DL/E65 | 220×110     | A4, сложенный втрое             |
| C7/C6  | 81×162      | A5, сложенный втрое             |
| C6     | 114×162     | A6 (или A4, сложенный вчетверо) |
| C6/C5  | 114×229     | A4, сложенный втрое             |
| C5     | 162×229     | A5 (или A4, сложенный вдвое)    |
| C4     | 229×324     | A4                              |
| C3     | 324×458     | A3                              |
| B6     | 125×176     | C6                              |
| B5     | 176×250     | C5                              |
| B4     | 250×353     | C4                              |
| E4     | 280×400     | B4                              |


## <span class="section-num">2</span> Российский стандарт {#российский-стандарт}

-   На территории Российской Федерации используется ГОСТ Р 51506-99, принятый 23 декабря 1999 года.
-   Этот стандарт опирается на ISO 269-85 в части размеров.

| Формат | Высота, мм | Ширина, мм | Примечание                   |
|--------|------------|------------|------------------------------|
| C6     | 114        | 162        | В СССР был основным форматом |
| DL/E65 | 110        | 220        | «Европейский» формат         |
| C5     | 162        | 229        |                              |
| C4     | 229        | 324        |                              |
| B4     | 250        | 353        |                              |
