{{< toc >}}


## <span class="section-num">1</span> Общая информация {#общая-информация}

-   Репозиторий: <https://github.com/argenkiwi/kenkyo>
-   Kenkyo --- это слоистая раскладка клавиатуры.
-   Разработана для стандартных ANSI-клавиатур.
-   Расширяет функциональность, сохраняя базовое поведение клавиатуры, что позволяет постепенно осваивать новые возможности без потери продуктивности.
-   Принципы
    -   Совместимость: не конфликтует со стандартной раскладкой.
    -   Эргономика: снижает нагрузку за счёт перераспределения функций.
-   Преимущества
    -   Уменьшает движение рук за счёт слоёв и комбинаций.
    -   Совместима с любыми ANSI-клавиатурами (включая компактные 60-65%).
    -   Поддерживает «аккорды» (например, W+E = Esc, X+C = Tab).
-   Есть готовые конфигурационные файлы для kanata (см. [Раскладка клавиатуры. kanata]({{< relref "2024-11-13-keymap-control-kanata" >}})) и keyd.


## <span class="section-num">2</span> Принципы дизайна {#принципы-дизайна}

-   Эргономика: Часто используемые клавиши (Esc, Tab) доступны без движения рук.
-   Совместимость: Не переопределяет стандартные сочетания (Ctrl+C, Ctrl+V работают как обычно).
-   Прогрессивность: Можно начать с базовых функций (Extend), постепенно осваивая Fumbol.


## <span class="section-num">3</span> Слои {#слои}

-   Main (основной): использует модификаторы на домашнем ряду (Home Row Modifiers) (см. [Клавиатура. Раскладка Home Row Mode]({{< relref "2025-04-25--keyboard-home-row-mode" >}})).
    -   Использует макет `GASC` / `◆⎇⇧⎈`.
-   Extend: активируется пробелом (см. [Клавиатура. Концепция SpaceFN]({{< relref "2025-05-09--keyboard-concept-spacefn" >}})), предоставляет доступ к навигации, медиа-клавишам и редактированию.
-   Fumbol: включает цифры на домашнем ряду, F1-F12 и дополнительные символы.


### <span class="section-num">3.1</span> Основной слой (Main) {#основной-слой--main}

-   Модификаторы на домашнем ряду (удерживаются для активации):
    -   A, S, D, F → Meta (Win/Command), Alt, Shift, Control.
    -   J, K, L, ; → Control, Shift, Alt, Meta (симметрично для правой руки).
-   Нижний ряд:
    -   Z, C, V → Control, AltGr, Fumbol (слой символов/чисел).
    -   M, ., / → Fumbol, AltGr, Control.
        -   Удерживая C (AltGr), можно ввести символы вроде € или ¶, если они назначены.

### <span class="section-num">3.2</span> Слой Extend {#слой-extend}

-   Активируется удержанием пробела (см. [Клавиатура. Концепция SpaceFN]({{< relref "2025-05-09--keyboard-concept-spacefn" >}})).
-   Правая рука:
    -   H, J, K, L → ←, ↓, ↑, → (навигация).
    -   U, I, O → Home, PgUp, End.
-   Левая рука:
    -   Q, W, E, R →
-   Media: громкость, пауза и т.д.
-   Пример: Удерживая пробел и нажав J, вы переместите курсор вниз.

{{< figure src="/ox-hugo/20250507T214500-extend.png" caption="<span class=\"figure-number\">&#1056;&#1080;&#1089;. 2.: </span>Слой Extend" >}}


### <span class="section-num">3.3</span> Слой Fumbol {#слой-fumbol}

-   Активируется через удержание V/M или через Extend + B:
-   Цифры на домашнем ряду:
    -   A, S, D, F, G → 1, 2, 3, 4, 5.
    -   H, J, K, L, ; → 6, 7, 8, 9, 0.
-   Пример: Удерживая V (Fumbol) и нажав S, вы введете цифру 2.
-   F-клавиши:
    -   Верхний ряд (Q-P) → F1-F10.
    -   B, N → F11, F12.

{{< figure src="/ox-hugo/20250507T214500-fumbol.png" caption="<span class=\"figure-number\">&#1056;&#1080;&#1089;. 3.: </span>Слой Fumbol" >}}


### <span class="section-num">3.4</span> Аккорды (Combos) {#аккорды--combos}

-   Комбинации двух клавиш для быстрых действий:
    -   W + E → Esc.
    -   I + O → Backspace.
    -   X + C → Tab.
-   Пример: Нажав X и C одновременно, вы вставите табуляцию.


## <span class="section-num">4</span> Пример рабочего процесса: {#пример-рабочего-процесса}

1.  Набираете текст → Используете основной слой.
2.  Нужно удалить слово → Удерживаете пробел (Extend) → Нажимаете I+O (Backspace).
3.  Вставить символ € → Удерживаете C (AltGr) → Нажимаете E.
4.  Переключиться на вкладку → Аккорд X+C (Tab).
