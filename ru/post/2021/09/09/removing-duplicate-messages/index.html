Почта. Удаление дубликатов сообщений из локальных почтовых ящиков

Обновлено 2023-09-14 4 мин. для прочтения Computer-Science

Удаление дубликатов сообщений из почтового хранилища типа Maildir. Для mbox тоже можно.

1 Программные средства

1.1 mail-deduplicate

  • Репозиторий: https://github.com/kdeldycke/mail-deduplicate
  • Сайт: https://mail-deduplicate.readthedocs.io/en/latest/
  • При первом поиске нашёл данную утилиту.
  • К сожалению, собственно функционал по удалению дубликатов писем продекларирован, но пока не реализован.

2 Реализация средствами командной строки

  • Принято решение реализовать на коленке.

2.1 Используемое программное обеспечение

  • Функционалом удаления почтовых дубликатов обладает утилита reformail.
  • Утилита является частью пакета maildrop, которая в свою очередь является частью почтового сервера Courier.
  • Сайт: https://www.courier-mta.org/maildrop/
  • Установка:
    • Linux
      • Gentoo
        1emerge maildrop

2.2 Предварительные сведения

  • Почтовый ящик maildir представляет собой дерево каталогов:

    1Maildir/cur
2Maildir/new
3Maildir/tmp

  • Программа reformail считывает сообщение со стандартного входа, обрабатывает его и записывает результат на стандартный вывод.

  • Извлечение и печать содержимого заголовка:

    1reformail -x

  • Обнаружение повторяющихся сообщений

    1reformail -D len filename

    • Параметр -D реализует удаления повторяющихся сообщений во входящей почте.

    • filename — файл кэша длиной примерно len байт.

    • Если сообщение имеет заголовок Message-ID:, который уже находится в файле кэша, повторное сообщение завершается с кодом выхода, равным 0.

    • В противном случае повторное сообщение завершается с кодом выхода, установленным на 1.

    • Например:

      1formail -D $((1024*1024*10)) /tmp/idcache
      • Кэш установлен в 10 мегабайт.

  • Для работы с форматом mbox используют параметр -s.

    • Файл почтового ящика в формате mbox разбивается на отдельные сообщения.
    • Для каждого сообщения выполняется внешняя программа.
    • Содержание каждого отдельного сообщения будет предоставлено внешней программе на стандартном вводе.

2.3 Реализация для Maildir

2.3.1 Удаление дубликатов

  • Простейший однострочник:
    1rm -f /tmp/idcache; for i in cur/*; do reformail -D $((1024*1024*10)) /tmp/idcache <$i && rm $i; done
  • Скрипт, удаляющий дубликаты во всех ящиках, синхронизируемых с помощью mbsync (см. Почта. Синхронизация. mbsync)
     1#!/bin/bash
 2# NAME: dedup-mailbox
 3
 4MAILDIR=~/Maildir
 5
 6list_imap=$(grep -e "^IMAPAccount" ~/.mbsyncrc | cut -d" " -f2 | xargs -I {} -n 1 echo {}" ")
 7
 8for i in ${list_imap}
 9do
10    list_imapdir=`ls ${MAILDIR}/${i}`
11
12    for j in $list_imapdir
13    do
14        cd "${MAILDIR}/${i}/${j}"
15
16        messages_before=`for k in cur/*; do [[ -f $k ]] && reformail -x Message-ID: <$k; done | wc -l`
17        messages_unique_before=`for k in cur/*; do [[ -f $k ]] && reformail -x Message-ID: <$k; done | sort -u | wc -l`
18
19        rm -f /tmp/idcache
20        for k in cur/*
21        do
22            [[ -f $k ]] && reformail -D $((1024*1024*10)) /tmp/idcache <$k && rm $k
23        done
24
25        messages_after=`for k in cur/*; do [[ -f $k ]] && reformail -x Message-ID: <$k; done | wc -l`
26        messages_unique_after=`for k in cur/*; do [[ -f $k ]] && reformail -x Message-ID: <$k; done | sort -u | wc -l`
27
28        echo "Number of messages in ${MAILDIR}/${i}/${j}"
29        echo -e "\t" "Total" "\t" "Unique"
30        echo -e "Before:\t" $messages_before "\t" $messages_unique_before
31        echo -e "After:\t"  $messages_after  "\t" $messages_unique_after
32    done
33
34done

2.3.2 Необязательные проверки

  • Перед и после удаления проверьте полное количество сообщений:
    1for i in cur/*; do reformail -x Message-ID: <$i; done | wc -l
  • Также проверьте количество уникальных сообщений:
    1for i in cur/*; do reformail -x Message-ID: <$i; done | sort -u | wc -l

2.4 Реализация для mbox

2.4.1 Удаление дубликатов

  • Скрипт, вызывающий программу обработки для каждого сообщения из mbox:
     1#!/bin/sh
 2# NAME: dedup-mbox
 3# INPUT: $1 = mbox
 4# OUTPUT: nmbox (файл с удалёнными дубликатами сообщений)
 5
 6# Delete temporary files
 7rm -f /tmp/tmpmail /tmp/idcache nmbox
 8
 9# reformail-mbox.sh is called for each mail
10cat $1 | reformail -s ./reformail-mbox.sh
  • Основной скрипт для дедупликации:
     1#!/bin/bash
 2# NAME: reformail-mbox.sh
 3# INPUT: stdin: a email
 4# called by dedup-mbox
 5
 6TM=/tmp/tmpmail
 7if [ -f $TM ]
 8then
 9    echo error!
10    exit 1
11fi
12cat > $TM
13
14# mbox format, each mail end with a blank line
15echo "" >> $TM
16
17cat $TM | reformail -D $((1024*1024*10)) /tmp/idcache
18
19# if this mail isn't a dup (reformail return 1 if message-id is not found)
20if [ $? != 0 ]
21then
22    # each mail shall have a message-id
23    if grep -q -i '^message-id:' $TM
24    then
25        cat $TM >> nmbox
26    fi
27fi
28
29rm $TM
