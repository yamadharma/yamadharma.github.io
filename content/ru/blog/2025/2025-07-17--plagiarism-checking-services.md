## <span class="section-num">1</span> Российские сервисы {#российские-сервисы}


### <span class="section-num">1.1</span> Антиплагиат {#антиплагиат}

-   Ссылка: <https://antiplagiat.ru/>
-   Основной сервис для проверки научных и учебных работ в РФ.
-   Используется большинством российских вузов (система «Антиплагиат.ВУЗ»).
-   Используется для проверки в научных журналах.
-   Проверка по закрытым базам: диссертации РГБ, «Кольцо вузов», научные публикации.
-   Есть бесплатный тариф (5 страниц, поиск только по интернет-источникам до 2021 года).


### <span class="section-num">1.2</span> Руконтекст {#руконтекст}

-   Ссылка: <https://incontext.ru/>
-   Разработанная при участии Института системного анализа РАН.
-   Проверка по базам: КиберЛенинка, НЭБ, Руконт, Википедия, база диссертаций РГБ.
-   Есть бесплатный доступ для студентов по ключу от вуза (ограниченное число проверок).


### <span class="section-num">1.3</span> Advego Plagiatus {#advego-plagiatus}

-   Ссылка: <https://advego.com/plagiatus/>
-   Раздельный анализ плагиата и рерайта.
-   Бесплатно для всех пользователей.


### <span class="section-num">1.4</span> Antiplagius {#antiplagius}

-   Ссылка: <https://antiplagius.ru/>
-   Агрегатор сервисов.
-   Проверяет текст в нескольких системах одновременно (Антиплагиат.ВУЗ, eTXT, Text.ru, Руконтекст).
-   Возможность повышения уникальности через кодировку или рерайт.
-   Бесплатные 20 000 символов для теста.


### <span class="section-num">1.5</span> Text.ru {#text-dot-ru}

-   Сайт: <https://text.ru/>
-   Дополнительные функции: SEO-анализ, проверка орфографии.
-   Бесплатная проверка до 15 000 символов в день.


### <span class="section-num">1.6</span> ETXT Антиплагиат {#etxt-антиплагиат}

-   Ссылка: <https://www.etxt.ru/antiplagiat/>
-   Проверка по поисковым системам (Google, Яндекс).
-   Глубокий анализ рерайта.
-   Бесплатный тариф: до 5 000 символов после регистрации.
