## <span class="section-num">2</span> Клавиатурные сочетания {#клавиатурные-сочетания}

-   Буфер поиска
    -   `g` : обновить представление списка фидов
    -   `G` : получить обновления фидов с серверов
    -   `s` : обновить фильтр поиска
    -   `c` : очистить фильтр поиска
-   Чтение
    -   `RET` : просмотр выбранной записи в буфере
    -   `b` : открыть выбранные записи в броузере (`browse-url`)
    -   `y` : скопировать URL выбранных записей в буфер обмена
    -   `r` : отметить выбранные записи как прочитанные
    -   `u` : отметить выбранные записи как непрочитанные
    -   `+` : добавить определенный тег к выбранным записям
    -   `-` : удалить определенный тег из выбранных записей
-   `q` : выход
