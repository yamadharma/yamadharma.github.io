Установка темы NexT для hexo

Обновлено 2020-12-02 1 мин. для прочтения sysadmin
Проект

Пробую поменять тему оформления блога на NexT.

Также можно посмотреть документацию:

  • https://github.com/theme-next/hexo-theme-next/blob/master/docs/ru/README.md
  • https://github.com/iissnan/hexo-theme-next/blob/master/README.md

Скачиваем тему

Переходим в каталог блога и скачиваем.

git clone https://github.com/theme-next/hexo-theme-next themes/next

Конфигурационный файл

mkdir -p source/_data
cd source/_data
touch next.yml

Убедимся, что в файле themes/next/_config.yml установлено:

override: false

В корневом файле _config.yml устанавливаем:

theme: next

и

source_dir: source

В файл source/_data/next.yml переносим из themes/next/_config.yml параметры, которые меняем.

Страницы категорий и тегов

Создадим страницы tags и categories

hexo new page "tags"
hexo new page "categories"

Отредактирем файлы source/tags/index.md

---
title: tags
date: "2018-04-04T14:41:32+00:00"
type: "tags"
---

и source/categories/index.md

---
title: categories
date: "2018-04-04T14:41:37+00:00"
type: "categories"
---

Добавим описание меню в файл source/_data/next.yml:

menu:
  home: / || home
  tags: /tags/ || tags
  categories: /categories/ || th
  archives: /archives/ || archive

Личная фотография

Сделаем каталог для изображений:

mkdir -p source/images

Поместим туда файл (например, avatar.jpg) и запишем в конфиг source/_data/next.yml:

avatar: /images/avatar.jpg

Я сделал ширину 126px.

Комментарии

Комментарии настроил через Disqus. Для этого сконфигурил на Disqus сайт с коротким именем yamadharma-blog. Добавил в файл source/_data/next.yml:

disqus:
  enable: true
  shortname: yamadharma-blog
  count: true
  lazyload: false
