## <span class="section-num">2</span> Список пакетов {#список-пакетов}


### <span class="section-num">2.1</span> array {#array}

-   CTAN: <https://ctan.org/pkg/array>
-   Позволяет определять новые стили колонок.
-   Может использоваться с другими пакетами для расширения их функционала.


### <span class="section-num">2.2</span> tabularx {#tabularx}

-   CTAN: <https://ctan.org/pkg/tabularx>
-   Расширение пакета tabular.
-   Позволяет создавать колонки с автоматически балансируемой шириной.
-   Вводит дополнительный тип колонок `X`.
-   Создаёт колонку, похожую на абзац, ширина которой автоматически расширяется так, чтобы заполнить объявленную ширину среды.


### <span class="section-num">2.3</span> tabulary {#tabulary}

-   CTAN: <https://ctan.org/pkg/tabulary>
-   Похож на _tabularx_.
-   Использует полную ширину таблицы в качестве аргумента.
-   Использует типы столбцов `L`, `C`, `R`, `J` для столбцов переменной ширины.
-   В отличие от столбцов `X` в _tabularx_, ширина каждого столбца взвешивается в соответствии с естественной шириной самой широкой ячейки в столбце.


### <span class="section-num">2.4</span> Пакет longtable {#пакет-longtable}

-   CTAN: <https://ctan.org/pkg/longtable>
-   Позволяет делать таблицы, которые продолжаются на следующей странице.
-   Столбцы на последовательных страницах имеют одинаковую ширину.
-   В отличие от _supertabular_ модифицирует процедуру вывода и, следовательно, не будет работать в многоколоночной среде.


### <span class="section-num">2.5</span> Пакет booktabs {#пакет-booktabs}

-   CTAN: <https://ctan.org/pkg/booktabs>
-   Открытые таблицы с дополнительными шпациями между строк.
-   Совместим с longtable.


### <span class="section-num">2.6</span> Пакет tabularray {#пакет-tabularray}

-   [LaTeX. Пакет tabularray]({{< relref "2025-07-13--latex_пакет_tabularray" >}})
-   CTAN: <https://ctan.org/pkg/tabularray>
-   Написан на LaTeX3.
-   Напрямую использует функции LaTeX3 для разбора таблицы.
-   Разделяет содержание и стиль таблицы.
-   Стиль таблицы может быть полностью задан с помощью ключей.


## <span class="section-num">3</span> Сравнение {#сравнение}

| Пакет        | Особенности                              | Когда использовать      |
|--------------|------------------------------------------|-------------------------|
| `tabular`    | Базовый синтаксис, минимум настроек      | Простые таблицы         |
| `array`      | Расширенные столбцы (p, m, b)            | Таблицы с текстом       |
| `tabularx`   | Автоматическая адаптация ширины          | Длинные таблицы         |
| `booktabs`   | Профессиональные линии                   | Научные публикации      |
| `tabularray` | Полный контроль через ключевые параметры | Сложные макеты, шаблоны |
