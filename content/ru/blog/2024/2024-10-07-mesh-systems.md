Mesh-системы.

<!--more-->

{{< toc >}}


## <span class="section-num">1</span> Общая информация {#общая-информация}

-   Предназначена для обеспечения непрерывного покрытия Wi-Fi.
-   Устройство, подключённое к маршрутизатору, является основным.
-   Остальные устройства (узлы) принимают и ретранслируют сигнал основного устройства.
-   Протоколы mesh-систем:
    -   802.11v позволяет гаджетам обмениваться информацией о состоянии сети, тем самым подбирая оптимальную точку доступа (должен поддерживаться клиентами);
    -   802.11k даёт точке доступа возможность оценить качество связи и перенаправить устройство на другую точку;
    -   802.11r сокращает время аутентификации, так как позволяет хранить ключи шифрования.


## <span class="section-num">2</span> Варианты систем {#варианты-систем}


### <span class="section-num">2.1</span> TP-Link Deco P9 {#tp-link-deco-p9}

-   Гибридного типа.
-   WiFi 5.
-   Скорость портов : 1 Гбит/с.
-   Количество портов Ethernet : 2.
-   Стандарты Wi-Fi : — 802.11a/b/g/n/ac.


### <span class="section-num">2.2</span> Huawei Wi-Fi Mesh 3 WS8100 {#huawei-wi-fi-mesh-3-ws8100}


#### <span class="section-num">2.2.1</span> Общая информация {#общая-информация}

-   Описание на сайте производителя: <https://consumer.huawei.com/ru/routers/wifi-mesh3/>
-   Инструкция: <https://manuals.plus/ru/huawei/ws8100-wifi-mesh-3-manual>
-   Обзор: <https://www.ixbt.com/nw/huawei-wifi-mesh-3-review.html>
-   4pda: <https://4pda.to/forum/index.php?showtopic=1043380>


#### <span class="section-num">2.2.2</span> Характеристики {#характеристики}

-   Входной интерфейс: 10/100/1000BASE-T
-   Частоты Wi-Fi: 2.4 ГГц, 5 ГГц
-   Стандарты Wi-Fi 802.11: a/ b/ g/ n/ ac/ ax
-   Класс Wi-Fi: AX3000
-   Порты: 10/100BASE-TX х 3, 10/100/1000BASE-TX х 3
-   Антенны: 4х внутренние


#### <span class="section-num">2.2.3</span> Особенности {#особенности}

-   На верхней панели расположена зона NFC для подключения мобильных устройств.
-   Каждый узел имеет по три гигабитных проводных порта. В зависимости от режима работы и варианта подключения, они выполняют функции WAN или LAN.


## <span class="section-num">3</span> Опыт использования {#опыт-использования}


### <span class="section-num">3.1</span> Huawei Wi-Fi Mesh 3 WS8100 {#huawei-wi-fi-mesh-3-ws8100}


#### <span class="section-num">3.1.1</span> Настройка {#настройка}

-   Предлагается использовать приложение _AI Life_.
    -   Для этого на основании устройства нанесён qr-код.
    -   Данное приложение дублирует (в упрощённой форме) функциональность web-интерфейса.
-   Адрес по умолчанию: 192.168.3.1
-   Система настраивает собственный NAT.
    -   Отключить его не удастся.
    -   То есть всегда настраивается как маршрутизатор.


#### <span class="section-num">3.1.2</span> Конфигурация по умолчанию {#конфигурация-по-умолчанию}

-   По умолчанию один модуль (который становится основным) подключён к маршрутизатору.
-   Остальные подключаются по WiFi.
-   При настройке основного модуля следует рядом (в радиусе 1 метра) разместить другие модули.
-   На них будет скопирована настраиваемая конфигурация.
