## <span class="section-num">1</span> Общая информация {#общая-информация}

-   CTAN: <https://ctan.org/pkg/extdash>


## <span class="section-num">2</span> Подключение пакета {#подключение-пакета}

-   Пакет добавляет несколько команд для тире в составных словах.
-   Загрузите этот пакет последним.
-   Для использования сокращений (например,  `\-/` вместо `\Hyphdash`):
    ```tex
    \usepackage[shortcuts]{extdash}
    ```
-   Разрешить перенос перед тире:
    ```tex
    \usepackage[allowbreakbefore]{extdash}
    ```


## <span class="section-num">3</span> Использование {#использование}


### <span class="section-num">3.1</span> Стандартные тире LaTeX {#стандартные-тире-latex}

-   `-` : стандартный дефис LaTeX
-   `--` : тире en-dash
-   `---` : длинное тире em-dash


### <span class="section-num">3.2</span> Разрывные тире extdash {#разрывные-тире-extdash}

-   Слова, написанные с помощью этих тире, могут также разрываться в других позициях, кроме тире.

| Команда      | Сокращение | Значение      |
|--------------|------------|---------------|
| `\Hyphdash`  | `\-/`      | дефис         |
| `\Endash`    | `\--`      | короткое тире |
| `\Emdash`    | `\---`     | длинное тире  |
| `\Halfspace` | `\-`       | пробел        |


### <span class="section-num">3.3</span> `extdash` неразрывные тире {#extdash-неразрывные-тире}

-   Перенос строки на месте дефиса невозможен.

| Команда       | Сокращение | Значение                  |
|---------------|------------|---------------------------|
| `\Hyphdash*`  | `\=/`      | неразрывный дефис         |
| `\Endash*`    | `\==`      | неразрывное короткое тире |
| `\Emdash*`    | `\===`     | неразрывное длинное тире  |
| `\Halfspace*` | `\=`       | неразрывный пробел        |
