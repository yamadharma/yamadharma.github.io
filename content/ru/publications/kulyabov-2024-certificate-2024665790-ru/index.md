abstract: Программа применяется для численного моделирования процесса передачи TCP-трафика,
  регулируемого алгоритмами RED, ARED, GRED, DSRED. Программный комплекс представлен
  функциями, отвечающими за численное моделирование, построение графиков и определение
  моделируемых систем для различных алгоритмов. Для реализации используются подключаемые
  библиотеки Plots.jl и DifferentialEquations.jl. В программе реализовано численное
  моделирование алгоритмов AQM для различных параметров системы. Помимо вычислений
  программа сохраняет графики поведения параметров системы для различных стартовых
  значений.
